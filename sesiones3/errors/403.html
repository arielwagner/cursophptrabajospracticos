403 Forbidden

El servidor no le permitirá a los visitantes acceder el archivo solicitado. Si un visitante ve este código de error, repase las configuraciones de permiso del archivo. También puede verificar si el archivo ha sido protegido mediante administrador de índice de cPanel.

Administración de Índice (Administrador de índice)

Para la versión 11.28 de cPanel

Cuando accede a un directorio (en vez de una página) en su sitio, un visitante tipicamente verá la página índice para ese directorio. Si no existe una página índice, el navegador mostrará una lista (o índice) de los archivos en ese directorio.

Al usar el Administrador de índice, es posible definir cómo verán un índice de directorio específico aquéllos que lo accedan. Puede desactivar el índice si no desea tener visible el contenido de un directorio.

  1. Seleccione el directorio que desea usar para navegar el contenido de su sitio web en la ventana emergente.
  2. Para navegar el Administrador de índice, pulse el icono de la carpeta junto al nombre del directorio.
  3. Pulse el nombre del directorio para el cual desea cambiar el estilo de índice.
  4. cPanel ofrece cuatro opciones; seleccione una de las seguientes:
    • Standard Indexing (texto solamente) (Índice estándar): El contenido sólo aparece como nombres de archivo.
    • Indexing elegante (gráficas y texto) (Índice elegante (gráficos y texto)): Información sobre los archivos, como tamaño y hora de la última actualización.
    • Configuración del sistema por defecto (Configuración del sistema predeterminada): Se utilizará la posición predeterminada definida por su antifrión web.
    • No Indexing (Sin índice): El contenido del directorio no está enumerado; los visitantes verán un mensaje que indica que el contenido está "prohibido".
  5. Pulse Salvar.
